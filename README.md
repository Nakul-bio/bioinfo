DNA Sequence Toolkit

This is a beginner friendly bioinformatics project written in python.

Overview

This toolkit performs basic DNA sequence analysis and demonstrates fundamental bioinformatics concepts using Python.
The project was built as a learning exercise to understand biological sequence processing and Python programming.

Features

DNA sequence validation

Nucleotide counting (A, T, G, C)

GC content calculation

Reverse complement generation

Transcription of DNA to RNA

Example

Input:
ATGCTGA

Output:
Valid DNA Sequence

Nucleotide Counts:

A: 2

T: 2

G: 2

C: 1

GC Content:
42.86%

Reverse Complement:
TCAGCAT

RNA Sequence:
AUGCUGA

Technology used: python3
